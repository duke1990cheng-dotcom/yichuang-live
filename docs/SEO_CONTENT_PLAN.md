# 住宅窗膜百科内容生产计划

本文件用于管理一窗官网「住宅窗膜百科」内容生产节奏。

## 生产规则

- 每次生产文章时，按照编号顺序生成下一篇。
- 新文章先进入「草稿」状态。
- 完成 SEO、GEO、FAQ、品牌落款检查后，进入「待审核」状态。
- 不要自动发布。
- 等待人工确认后再发布。
- 发布后将状态更新为「已发布」。

## 内容要求

每篇「住宅窗膜百科」文章必须包含：

- SEO 优化标题
- SEO Description
- GEO 友好开头：开头直接回答用户核心问题
- FAQ 结构
- 一窗生活科技品牌落款
- 不夸大效果，不写无法验证的数据
- 不使用「100%」「保证」「行业第一」等夸张表达
